# I Like Puzzles

I really enjoy doing my weekly Sudoku when the newspaper is dropped off at my apartment. 
I created this simple solving algorithm as an expression of my love of Sudoku and a somewhat as a distillation of my limited solving wisdom.
I think when I was writing this code up I was watching [Cracking the Cryptic](https://www.youtube.com/c/CrackingTheCryptic) near daily.
